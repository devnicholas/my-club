Sobre o MyClub

O MyClub é um projeto que permite o armazenamento e o gerenciamento de algumas informações relacionadas a um clube de futebol. É voltado principalmente para o Modo Carreira do FIFA.

Com o objetivo de levar mais seriedade e realismo para o game, sempre registrei dados sobre os jogos, jogadores, adversários, transferências, etc, tudo pra conseguir gerenciar meu time do FIFA o mais próximo possível da realidade de um time de futebol profissional.

A plataforma MyClub foi criada como forma de centralizar e digitalizar o processo de salvamento desses dados, além de proporcionar para outros jogadores a mesma experiência que busquei para mim.

Armazenamento de informações

O MyClub armazena todas as informações apenas no navegador em que está sendo executado, não sendo possível coletar ou consultar nenhum dado pela equipe do MyClub ou qualquer outro usuário.

Para compartilhar as informações entre dispositivos é necessário exportar os dados e então importar no dispositivo onde desejar. Isso tambem vale para diferentes navegadores no mesmo dispositivo.
